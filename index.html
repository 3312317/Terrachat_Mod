Core Features

专注于玩家之间的聊天

系统提示、世界事件和其他非玩家消息不会进入 TerraChat，记录内容保持干净。

◈

社交式聊天 UI

本人消息在右，其他玩家在左；头像、名称、正文和时间清晰分层。

⌨

输入法优化

支持中文候选框、光标、文本选择、长按退格与常用 Ctrl 快捷键。

◎

本地图片头像

选择电脑图片，拖动位置、滚轮缩放并裁切成统一规格的圆形头像。

◇

玩家资料卡

设置资料名和 200 字描述；资料名旁保留原角色名，避免认错玩家。

▣

聊天记录管理

最多保存 200 条，可删除单条消息，也可通过确认弹窗清空全部。

↻

多人资料同步

头像、资料名与描述会同步给其他 TerraChat 玩家，聊天时间由服务器统一。

Input Experience

面板内直接输入与发送

输入逻辑沿用经过反复调整的稳定规则，适合连续聊天和较长文本编辑。

  • Enter 发送，Shift+Enter 或 Ctrl+Enter 换行
  • Ctrl+A / C / X / V 全选、复制、剪切和粘贴文字
  • 方向键、Home、End、Delete 与长按退格
  • 点击输入框外取消焦点，快捷键不会误输入
Avatar & Profile

圆形头像与个人资料

点击顶部头像或聊天头像即可打开资料卡；自己的资料可编辑，其他玩家保持只读。

  • GPU 实时裁切预览，拖动与缩放更加流畅
  • 统一头像规格与高清显示纹理
  • 文件选择器记住上次使用的图片目录
  • 资料名显示原角色名，描述最多 200 字
History

聊天记录跟随角色保存

退出地图或游戏后不会消失。不同角色分别保存自己的记录、资料名与个人描述。
