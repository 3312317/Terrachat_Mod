TerraChat 不替换 Terraria 的聊天系统。它保留原版发送、指令与多人联机方式，只把真实玩家发送的消息重新整理成带头像、资料名和时间的聊天记录。

  • 按角色保存记录、资料名与描述
  • 最多 200 条可单条删除或全部清空
  • 默认按 C可在按键设置中修改

Made for Terraria

不是套一层聊天皮肤

界面、输入和多人同步都围绕 Terraria 的实际使用场景调整，同时尽量不改变玩家已经熟悉的原版操作。

01 / VANILLA CHAT

继续使用原版发送方式

消息仍通过 Terraria 原版聊天通道发送，常用聊天指令和多人联机规则不需要重新学习。

02 / PLAYER ONLY

记录只保留玩家发言

世界事件、系统提示和 NPC 对话不会混进记录，打开面板时看到的是玩家之间的交流。

03 / TERRARIA UI

适配游戏内界面层级

面板适配不同分辨率与 UI 缩放；原版对话位于面板下方，鼠标光标保持在最上层。

04 / LONG TEXT

中文输入与长文本编辑

支持输入法候选框、选择与粘贴、多行光标移动，以及输入框独立滚动和边界裁剪。

05 / MULTIPLAYER

多人资料与统一时间

同服 TerraChat 玩家之间同步资料名、描述和头像；多人记录时间以服务器时间为准。

06 / LOCAL CONTROL

记录由当前玩家管理

可以删除本地单条记录或确认后全部清空；这只是整理自己的历史，不是撤回服务器消息。

Save boundaries

哪些跟角色，哪些全角色共用

TerraChat 有两种保存范围。这里明确列出，避免换角色后产生误解。

按角色保存

每个角色各有一份

同一个角色进入不同世界时仍沿用自己的数据；切换角色后则读取另一个角色的数据。

  • 最多 200 条玩家聊天记录
  • 资料名
  • 最多 200 字的个人描述
本机全角色共用

头像不是角色独立的

头像图片保存在 TerraChat 的本地配置中。本机所有角色使用同一张头像；更换头像后，其他角色下次打开也会使用新头像。

  • 选择本地图片并裁切为圆形
  • 进入多人游戏时随当前资料同步

资料名和描述跟随角色，头像跟随本机 TerraChat 配置。删除角色聊天记录不会删除头像文件。

Input guide

输入框按键说明

按键行为以多行文本编辑为准。Home 和 End 操作的是整段输入内容，不是聊天记录。

Enter

发送消息。 输入为空时不会发送。

Shift+Enter/Ctrl+Enter

插入换行。 用于编写多段或较长消息。

↑↓

在可见文本行之间移动光标。 按住可以连续移动，Shift 可扩展选择。

←→/Ctrl

按字符移动光标。 同时按 Ctrl 时按词移动。

HomeEnd

Home 跳到全文开头，End 跳到全文末尾。 配合 Shift 可从当前位置选择到开头或末尾。

Ctrl+ACtrl+CCtrl+XCtrl+V

全选、复制、剪切与粘贴。 支持多行剪贴板文字。

滚轮拖动

滚动长输入内容。 输入框和聊天记录各自拥有独立滚动区域。

Before using

使用前需要知道

这些边界属于当前版本的设计，不会在后台额外创建账号或云端聊天服务。

会替换原版聊天吗？

不会。TerraChat 是额外的记录与输入面板，消息仍走原版聊天系统。

为什么看不到系统提示？

当前版本只收集真实玩家发送的消息，其他内容继续留在原版界面。

删除记录等于撤回吗？

不是。删除只影响当前角色本地保存的 TerraChat 历史。

别人能看到哪些资料？

同服 TerraChat 玩家可以看到同步后的头像、资料名和描述；原角色名仍会保留用于辨认。

进入世界后，按 C 打开 TerraChat 默认快捷键可在 tModLoader 的按键设置中重新绑定。
